# a8k

集成 webpack 的 react 项目最佳实践配置的构建工具

## 使用方法

安装

```shell
npm i -g a8k
```

生成模板

```shell
k create [projectName]
```

启动开发服务器

```shell
k dev
```

dev 调试直出模式

```shell
k dev -s
```

构建前端代码

```shell
k build
```

运行 jest 测试

```shell
k test
```
